Admin Login

Administrator access portal. Multiple authentication vulnerabilities present.

⚠️ Authentication Vulnerabilities

  • Default Credentials: admin / admin (or any password)
  • SQL Injection: Username: admin'-- (bypasses password)
  • No Rate Limiting: Vulnerable to brute force attacks
  • Weak Session Management: Session tokens in URL parameters
  • No 2FA/MFA: Single factor authentication only
  • Verbose Error Messages: Reveals if username exists
  • No Account Lockout: Unlimited login attempts
  • Password in GET Request: Passwords logged in server logs

Attack Vectors:

  • SQL Injection: admin' OR '1'='1'--
  • NoSQL Injection: {"$ne": null}
  • LDAP Injection: admin)(|(uid=*
  • Default Credentials: admin / admin
  • Session Fixation: Manipulate session tokens
